# 线性表

## 一.顺序表

### 1.头文件及顺序表的定义

```c++
#include <iostream>
using namespace std;
typedef int Elemtype;
#define MAXSIZE 100000
class Sqlist//定义顺序表
{
public:
	Elemtype*m_Elem;//动态存储空间
	int m_Length;
};
```

### 2.初始化顺序表

```c++
bool Init_Sqlist(Sqlist&L)//初始化线性表
{
	L.m_Elem = new int[MAXSIZE];//开辟内存空间
	if (!L.m_Elem)
	{
		return false;
	}
	L.m_Length = 0;
	return true;
}
```

### 3.摧毁和清空顺序表

```c++
void Destory_Sqlist(Sqlist&L)//摧毁线性表
{
	L.m_Elem = NULL;
	delete L.m_Elem;
}
bool Clear_Sqlist(Sqlist&L)//清空线性表
{
	if (!L.m_Elem)
	{
		return false;
	}
	L.m_Length = 0;
	return true;
}
```

### 4.取顺序表第i个位置

```c++
int& Get_Elem(Sqlist&L, int i, Elemtype&e)//查找值的位置
{
	for (int i = 0; i < L.m_Length; i++)
	{
		if (e == L.m_Elem[i - 1])
		{
			return i;
		}
	}
}
```

### 5.在i处插入值e

```c++
bool Add_Elem(Sqlist&L, int i, Elemtype&e)//在位置i处插入数值e
{
	if (i<1 || i>L.m_Length + 1||L.m_Length>=MAXSIZE)//异常处理
	{
		return false;
	}
	L.m_Length++;
	for (int j = L.m_Length - 1; j >= i; j--)
	{
		L.m_Elem[j + 1] = L.m_Elem[j];
	}
	L.m_Elem[i-1] = e;
}
```

### 6.在i处删除数据

```c++
bool Delete_Elem(Sqlist&L, int i)//在位置i处删除数据
{
	if (i<1 || i>L.m_Length  || L.m_Length > MAXSIZE||L.m_Length==0)//异常处理
	{
		return false;
	}
	for (int j = i; j < L.m_Length - 1; j++)
	{
		L.m_Elem[j-1] = L.m_Elem[j];
	}
	L.m_Length--;
}
```
